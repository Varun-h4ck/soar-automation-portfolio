# SOAR Automation Portfolio

A collection of security automation workflows
built with Shuffle SOAR, VirusTotal, Wazuh,
and TheHive — designed to demonstrate SOC
analyst and security automation skills.

## Lab Setup

| Tool | Type | Purpose |
|------|------|---------|
| Shuffle SOAR | Cloud | Automation engine |
| VirusTotal | API | Threat intelligence |
| Wazuh | Self-hosted VM | SIEM + EDR |
| TheHive | Self-hosted VM | Case management |
